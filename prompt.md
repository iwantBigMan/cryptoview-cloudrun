# 작업 프롬프트 기록

## 기능 관련 프롬프트
- Gate.io 평균단가 API가 `currencyPair=SOL_USDT` 요청에 대해 `averagePrice=0`, `tradeCount=0`으로 응답하는 문제를 확인했다.
- 체결 내역이 없을 때 0으로 계산되는 구조를 분석하고, 기간 미지정 요청에서는 최근 1년 체결 내역을 30일 단위로 나눠 조회하도록 변경했다.
- Android 요청 형식은 유지하고, 백엔드에서 기본 조회 범위를 보완하는 방식으로 구현했다.

## 테스트 관련 프롬프트
- 변경 후 TypeScript 빌드를 실행해 컴파일 오류가 없는지 확인했다.

## 기타 프롬프트
- `dist` 폴더는 직접 수정하지 않고 `src` 기준으로만 변경했다.
- 민감 정보는 로그에 출력하지 않는 기존 보안 원칙을 유지했다.
